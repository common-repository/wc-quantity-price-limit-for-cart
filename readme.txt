== Installation ==

1. Upload the plugin files to the `/wp-content/plugins` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to WooCommerce > Quantity & Price Limit to configure the plugin.

== Frequently Asked Questions ==

= How do I set the quantity and price limits? =

After activating the plugin, navigate to WooCommerce > Settings > Quantity & Price Limit. From there, you can set the limits for individual products and for the entire cart.

= Is this plugin compatible with the latest version of WooCommerce? =

Yes, Quantity & Price Limit for WooCommerce Cart is fully compatible with the latest version of WooCommerce.

= Can I set different limits for different products? =

Yes, you can set different quantity and price limits for each individual product.

= Can I exclude certain products from the global rules? =

Yes, the plugin allows you to exclude specific products from all minimum and maximum rules. Additionally, you can override global settings with local adjustments for individual products.

= Can I set limits for the entire cart, not just individual products? =

Yes, the plugin allows you to set overall minimum and maximum quantities for the entire cart, ensuring customers adhere to your defined purchase limits for their entire order.

= What happens if I disable category limits for a specific product? =

If you disable category limits for a specific product, the category rules will not apply to that product. This allows you to have more flexibility and control over individual product settings, independent of the category rules.

= What is the quantity step feature, and how do I use it? =

The quantity step feature allows you to define the increments by which quantities increase. For example, setting a step of 2 means customers can only purchase in multiples of 2 (e.g., 2, 4, 6). This is particularly useful for products sold in packs. To use this feature, adjust the quantity step setting in the individual product settings.

= Will the plugin restrict customers from purchasing more than the maximum quantity? =

Yes, the WC Quantity & Price Limit for Cart plugin will enforce the maximum quantity limits you set. Customers will not be able to add more than the specified maximum quantity to their cart, ensuring that your sales rules are followed.

= Can I set quantity limits across my entire store or only for specific products? =

You can set quantity limits both globally across your entire store and for specific products. This flexibility allows you to manage purchase limits effectively based on your store's requirements.

= What if I do not set the cart rule? =

If you do not set cart rules, the cart will not enforce any minimum or maximum limits. Customers will be able to purchase any quantity and total value allowed by the individual product and category settings.
